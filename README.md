- hx_MeterRobbery -
A robbery script to rob parking meters

Dependencies --
1. ox_inventory
2. ox_lib
3. ox_target
4. ESX
5. cd_dispatch

Install --
1. Add the following;
	['screwdriver'] = {
		label       = 'Screwdriver',
		weight      = 100,
		stack       = true,
		close       = false,
		description = 'Bob the builder, can he fix it?'
	},
to ox_inventory > data > items.
2. Add the image in hx_MeterRobbery > _INSTALL > images to ox_inventory > web > images
3. Enjoy!
